# Books Scraper

Парсер для сбора данных о книгах с сайта [Books to Scrape](http://books.toscrape.com/).

## Описание

Проект представляет собой веб-скрапер, который собирает информацию о книгах:
- Название книги
- Рейтинг
- Цена
- Наличие на складе
- UPC код
- и другие характеристики

## Функциональность

- **`scrape_book_data(url)`** - парсит данные одной книги по URL
- **`scrape_books(save_to_file=False)`** - парсит все книги с сайта (1000 книг)
- Автоматическое сохранение в JSON формате
- Поддержка пагинации
- Обработка ошибок и таймаутов
